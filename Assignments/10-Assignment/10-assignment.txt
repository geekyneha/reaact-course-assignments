Ways to Write CSS in React

1. Inline Styles

   ```js
   <div style={{ color: 'red' }}>Hello</div>
   ```

2. Normal CSS / External Stylesheets.

    `style.css`
    Import using: `import './style.css';`

3. CSS Modules

    Locally scoped styles
    `import styles from './Button.module.css'`

4. CSS Preprocessors

   Sass / SCSS

     Write nested, modular CSS with variables and mixins.
     Requires `node-sass`.

5. CSS-in-JS (Libraries)

    `styled-components`
    `emotion`

6. Component Libraries

   * Material UI
   * Chakra UI
   * Ant Design
   * Lucid UI
   * ShadCN UI (based on Tailwind)

7. Utility-first CSS Frameworks

   * Tailwind CSS
   * Bootstrap

 ‚öôÔ∏è How to Configure Tailwind CSS

1. Install dependencies:

```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init
```

2. **Configure `tailwind.config.js`**:

```js
content: ["./src/**/*.{js,jsx,ts,tsx}"],
theme: { extend: {} },
plugins: [],
```

3. **Create or configure `postcss.config.js`**:

```js
module.exports = {
  plugins: {
    tailwindcss: {},
    autoprefixer: {},
  },
};
```

4. **Import Tailwind in your `index.css`:**

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

---

### üõ†Ô∏è **tailwind.config.js Keys Explained**

| Key       | Purpose                                                        |
| --------- | -------------------------------------------------------------- |
| `content` | Tell Tailwind where to scan for class names (tree shaking)     |
| `theme`   | Define default design tokens like spacing, fonts, colors       |
| `extend`  | Add custom values **without replacing defaults**               |
| `plugins` | Add 3rd-party or custom Tailwind plugins like forms/typography |

---

### üìÑ **Why do we need `postcss.config.js`?**

* Tailwind works with **PostCSS** to transform styles.
* This file:

  * Registers **Tailwind and Autoprefixer**
  * Allows **plugin configuration**
  * Helps in **purging unused styles**
  * Keeps your CSS **light and optimized**
